[Linux] zsh 설치
2021. 8. 11.

리눅스에서는 터미널을 많이 사용한다. 이 때 명령을 해석하고 실행시키는 것이 shell이다. 우분투에서는 기본 쉘로 bash를 사용한다.

zsh는 bash보다 기능도 많고 플러그인이나 테마를 이용해 커스터마이징이 가능해서 기본 쉘보다 더 예쁘고 편리하게 터미널을 사용할 수 있도록 해준다. 이를 위해서는 oh-my-zsh도 같이 설치해야 한다.

oh-my-zsh 설치




1. zsh 설치

현재 사용 중인 쉘을 확인해보자.

$ echo $SHELL
/bin/bash

기본 쉘은 bash이다.


이제 zsh를 설치한다.

$ sudo apt-get install zsh

2. zsh를 기본 쉘로 변경

zsh가 설치된 위치를 확인한다.

$ which zsh
/usr/bin/zsh

확인한 위치를 넣어 기본 쉘을 변경한다.

$ chsh -s /usr/bin/zsh

그러고 터미널을 다시 시작하니 이런 화면이 떴다.

  • (q) : 그냥 종료한다. 다음에 이 화면이 다시 뜬다.
  • (0) : ~/.zshrc 파일을 생성만 하고 종료한다. 이 화면은 다시 뜨지 않는다.
  • (1) : 메인 메뉴로 들어간다.
  • (2) : ~/.zshrc 파일을 생성하고 추천하는 기본값으로 세팅한다.

나는 (2)를 선택했다. 쉘이 이렇게 바뀌었다.


다시 사용 중인 쉘을 확인해보면 zsh가 잘 나온다.

$ echo $SHELL
/usr/bin/zsh
